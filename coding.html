Coding

This website is 66% HTML and 34% CSS. In this webpage, I have implemented my following skills :

  • Utilized semantic HTML elements such as header, nav, main, footer, and form for better document structure and accessibility
  • Incorporated FontAwesome icons library for social media icons, enhancing visual appeal and functionality.
  • Added a custom favicon to enhance the webpage's branding and visual identity.
  • Included meta tags for defining character set, viewport settings, page title, description, and keywords for SEO optimization.
  • Ensured that the webpage layout adjusts appropriately to different screen sizes and devices using CSS media queries. For instance, when you open the same page on a mobile phone certain elements will not show. I have successfully implemented media queries for tablet phones and mobile phone.
  • Integrated a video element to display a self-introduction video, enhancing user engagement and experience.
  • Applied inline CSS styles for specific elements within the HTML code for quick styling adjustments.
  • Imported external CSS files (resume.css) to maintain separation of concerns and manage styles efficiently.
  • Used CSS classes to style elements, such as .site-header, .socialmedia, .themonic-nav, .wrapper, .site-content, and .list-of-skills.
  • Designed and implemented a form on the contact page using various input elements form controls such as submit button , textarea element, dropdown list.
  • Applied HTML5 form validation attributes such as required, pattern, and placeholder for ensuring data integrity and guiding user input.
  • Configured the form to submit data via email using the mailto attribute in the form's action attribute, enabling users to send messages directly from the webpage.
  • Utilized inline JavaScript for a simple form submission confirmation alert (onclick="alert('Thankyou for filling the form,see you soon!')") upon clicking the submit button.
  • Ensured accessibility by providing alternative text for images (alt attribute) and semantic markup for screen readers and assistive technologies.
